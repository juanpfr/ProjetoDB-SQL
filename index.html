Projeto Banco de Dados - Universidade

Este projeto apresenta a modelagem de um banco de dados completo para uma universidade fictícia. A estrutura envolve entidades como Aluno, Professor, Curso, entre outras, com regras de negócio, normalização e comandos SQL organizados para facilitar a compreensão e implementação.
